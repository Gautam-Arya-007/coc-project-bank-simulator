# coc-project-bank-simulator

Title: Student Score Analyzer
<br>
Description:
<br>
A C program that helps analyze quiz scores of students.
<br>
It takes the number of students and their marks as input, then calculates the average, highest, and lowest scores, and classifies each student’s performance (Excellent, Good, Average, or Needs Improvement).

Concepts Used
<br>
C Concepts:
<br>
1D Arrays
<br>
for loops
<br>
if–else statements
<br>
Functions
<br>
Variables and I/O (printf, scanf)

Math Concepts:
<br>
Mean (Average)
<br>
Minimum and Maximum values

How to Compile
<br>
gcc student_score_analyzer.c -o analyzer -lm

How to Run
<br>
./analyzer

Example:
<br>
Input student marks → Get average, highest, and lowest scores with performance summary.
